Events

The Events system is the most actively used part of the site. It powers the public calendar, the homepage upcoming-events widget, and the event detail pages where members can view schedules and results. This guide covers what members see and how to create new events.

Public Calendar (What Members See)

https://www.okanoganvalleygolf.com/events

The public calendar shows all upcoming events on a month-by-month grid. Each event is color-coded by category. Members can:

  • Browse forward and backward through months.
  • Click any event to open its detail page (time, description, results if posted).
  • See cancelled events displayed with a strikethrough badge (they remain visible so members know the event was planned).

Admin Event List

https://www.okanoganvalleygolf.com/admin/events

This is your starting point for all event management. The list shows every event in the system along with its category, start date, recurrence status, and current status (active / cancelled).

From each row you can:

  • Edit — change any field (affects all occurrences of a recurring series).
  • Results — post or update post-event results.
  • Cancel — cancel from a specific date forward, or the entire event.
  • Restore — un-cancel a previously cancelled event.
  • Delete — permanently remove the event and all its data.

Create a New Event

https://www.okanoganvalleygolf.com/admin/events/create

Click Create Event from the admin event list. The form is divided into four sections: basic info, date & time, recurrence, and blackout dates.

Basic Info

  1. Enter a Title (required) — this is what members see on the calendar and detail page.
  2. Choose a Category — controls the color on the calendar. See the color table above.
  3. Optionally add a Description — include eligibility rules, handicap limits, contact info, entry fees, or any other notes members need to know.

Date & Time

  1. Check All Day Event if the event occupies the whole day (no specific start/end time shown — typical for course-closed days).
  2. Set Start Date and, unless it's all-day, Start Time.
  3. Set End Date and End Time. For a single-day event, End Date = Start Date.

For recurring events the start and end date/time define a template for each occurrence. Every occurrence will use the same time of day as the original start. You do not need to re-enter it each week.

Recurring Events

Check Recurring Event in the Recurrence section to reveal the recurrence options. The three repeat frequencies are:

Every Day

Select Every Day from the Repeat dropdown. No additional day selection is needed. Useful for multi-day course closures.

Every Week

Select Every Week and then tick one or more days of the week. For example, ticking Wed creates an event every Wednesday. Ticking Tue and Thu creates two occurrences per week.

The start date you set determines the first occurrence. The system will only generate occurrences on the days you ticked, starting from that date.

Every Month

Select Every Month and enter a day pattern in the On field. This uses a coded format:

CodeMeaningExample
1SA1st Saturday of each monthMember-guest tournament on 1st Saturday
2TU2nd Tuesday of each monthBoard meeting on 2nd Tuesday
-1MOLast Monday of each monthEnd-of-month league night
3WE3rd Wednesday of each monthLadies' day on 3rd Wednesday

The monthly day code is case-sensitive and must use the two-letter abbreviations: MO TU WE TH FR SA SU. A typo here will cause the event to not repeat correctly.

Repeat Until

Leave blank to repeat indefinitely (you can always cancel later). Set a date to stop the series automatically on that date — useful for a summer league that ends in August.

Best practice: For seasonal events (leagues, school programs) always set a Repeat Until date matching the end of the season. This prevents orphaned future occurrences appearing on the calendar after the season ends.

Blackout Dates

Blackout dates silently remove specific occurrences of a recurring event without cancelling the whole series. They do not appear on the public calendar at all — the date simply has no event.

Use blackout dates for:

  • A holiday week when the course closes.
  • A one-off date when a regular event won't be held but will continue the following week.
  • Tournament prep days that conflict with a recurring league.

To add a blackout date:

  1. In the Blackout Dates section, click the date picker and choose the date to skip.
  2. Click Add Date. It appears as a removable tag.
  3. To remove a blackout date, click the × on its tag.
  4. Save the form.

Blackout vs. Cancel: A blackout date removes the occurrence silently — members won't see any trace of it. A cancellation shows the event on the calendar with a "Cancelled" badge. Use blackout when the event simply doesn't happen; use Cancel when you want members to know it was planned but called off. See the Managing Events guide for how to cancel.
