Board Minutes

Use this page to upload, update, and remove Board of Directors meeting minutes. Minutes are stored as PDF files and organized by meeting date. All management is done from the admin panel — there is no public listing page; PDFs are accessed directly via their file URL.

Getting to the Board Minutes Admin

  1. Log in and go to the admin dashboard.
  2. Click Board Minutes in the admin navigation, or go directly to:
https://www.okanoganvalleygolf.com/admin/board-minutes

This page lists all uploaded minutes in reverse chronological order (most recent first), with each row showing the meeting date, the filename, and Edit/Delete buttons.

Screenshot suggestion: /admin/board-minutes showing the minutes table with date, filename, and action buttons for at least two entries.

Upload New Minutes

  1. From /admin/board-minutes, click Upload Minutes.
  2. Fill in the form:
    • Meeting Date — The date of the board meeting (required).
    • PDF File — The minutes document (required). Must be a PDF; max 10 MB.
  3. Click Upload.

The file is saved to the server and the entry appears in the minutes list sorted by date.

Screenshot suggestion: the Upload Minutes form showing the Meeting Date field and PDF file selector.

The filename stored on the server is generated automatically to avoid conflicts — the original name of the file you select is not used. The meeting date is what identifies the entry in the list.

Edit an Entry

You can change the meeting date or replace the PDF for an existing entry.

  1. Click Edit next to the entry.
  2. Update the Meeting Date if needed.
  3. To replace the PDF, select a new file under Replace PDF. Leave it blank to keep the existing file.
  4. Click Update.

The current file is shown as a link on the edit screen so you can open it to confirm it is correct before deciding whether to replace it.

Delete an Entry

  1. Click Delete next to the entry and confirm.

Both the database record and the PDF file on the server are removed immediately. There is no undo.

Deletion is permanent. If the PDF is linked from elsewhere on the site (e.g., a menu item or page), that link will break. Download the file to your computer first if you may need it later.

Tips & Gotchas

Use the actual meeting date, not the upload date.
Minutes are sorted by meeting date. Using today's date instead of the date of the meeting will put the entry in the wrong position in the list.

PDFs only, 10 MB max.
The uploader rejects any file that is not a PDF or exceeds 10 MB. If your document is too large, try exporting it from Word or your PDF editor at a lower quality setting to reduce the file size.

Duplicate dates are allowed.
Nothing prevents uploading two entries with the same meeting date (e.g., a draft and a final version). Use the Edit screen to correct or remove duplicates.
