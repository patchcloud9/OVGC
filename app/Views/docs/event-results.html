Event Results

After a tournament or league round completes, you can post results directly on the event's detail page. Members visiting /events can click the event and see the leaderboard, scores, and course condition notes.

What Members See

When results have been posted for an event, the public detail page shows:

  • A Results section with the formatted leaderboard text.
  • An optional Course Conditions / Notes section below the scores.
  • The date the results were last updated.

If no results have been posted yet, the Results section is not shown.

Screenshot suggestion: public event detail page with a posted leaderboard visible.

Getting to the Results Form

One-Time Event

  1. Go to /admin/events.
  2. Click Results next to the event.
  3. The results form opens immediately — there is only one occurrence to post for.

Recurring Event

A recurring event (e.g., weekly league) has many occurrences. You pick the specific date you want to post results for:

  1. Go to /admin/events.
  2. Click Results next to the recurring event.
  3. A date picker page lists all occurrences, newest first. Dates that already have results posted are marked with a trophy icon.
  4. Click a date to open the results form for that occurrence.
Screenshot suggestion: occurrence picker page showing a list of dates with some marked as "Results posted".

You can post results for any past or future occurrence. Results for future dates are not shown on the public site until that date has passed.

Posting Results

The results form has two fields: Results / Leaderboard and Course Conditions / Notes.

  1. Enter the leaderboard in the Results / Leaderboard box using the formatting described below.
  2. Optionally enter weather, course conditions, or attendance notes in the Course Conditions / Notes box.
  3. Click Post Results.

Results save immediately and appear on the public detail page right away.

Formatting the Results Text

The results field uses a simple formatting system. No HTML knowledge is needed.

What you typeHow it appears on the site
**1st Place:**1st Place: (bold text)
--Low gross winner• Low gross winner (bullet list item)
A blank lineParagraph break / visual spacing
Press Enter onceNew line within the same paragraph

Example Results Entry

**1st Place:** John Smith - 72
**2nd Place:** Jane Doe - 74
**3rd Place:** Bob Johnson - 75

-- Low Gross: John Smith (72)
-- Low Net: Jane Doe (66 net)
-- Closest to Pin #7: Bob Johnson

**Skins results:**
-- Hole 3: Jane Doe
-- Hole 9: John Smith
-- Hole 14: No winner

This would render with bold labels, bullet points for the side awards, and clear visual groupings.

Keep it simple. Most members just want to quickly see where they placed and who won skins. A clean list with bold labels for each place and bullets for side awards is easy to read on both desktop and mobile.

Editing Results

If you need to correct a score or add a missed entry, just navigate back to the same results form and update the text. The form will show the current saved content pre-filled in the text area.

  1. Navigate to the event results form the same way you did when posting (via the Results button in the admin list).
  2. Edit the text in the Results box.
  3. Click Update Results.

The "last updated" timestamp shown to members will refresh to the current time.

Removing Results

If results were posted by mistake or need to be completely cleared:

  1. Navigate to the results form for the event and date.
  2. Click the red Remove Results button.
  3. Confirm the deletion in the browser prompt.

The Results section will no longer appear on the public detail page for that occurrence. The event itself is unaffected.

Removing results is permanent — the text cannot be recovered. If you need to make corrections, use Update Results instead of removing and re-entering.

View the Public Page After Posting

The results form has a View on Site button that opens the public event detail page in a new tab. Use this to confirm formatting looks correct after saving.

You can also navigate directly to the public page:

  • One-time event: /events/{id}
  • Recurring event occurrence: /events/{id}/{YYYY-MM-DD}

Tips

Post within a day or two. Members check results quickly after an event. Posting within 24 hours keeps engagement high and reduces "did anyone post results yet?" questions.

Use Course Conditions for context. Noting "Greens were aerated, scores ran high" or "9-hole round only due to weather" helps members understand the scores in context, especially for handicap calculations.

Bold formatting uses double asterisks. A common mistake is using single asterisks (*text*) which won't render as bold. Always use double: **text**.

Bullet items need two dashes. Single dashes won't create a bullet. Use -- at the start of the line: -- Low gross: John Smith.
