Board of Directors

Use this page when you need to add, update, or remove members from the Board of Directors listing at /board-members. Each entry can include a name, board title, contact email, and an optional headshot photo.

Getting to the Board Admin

  1. Log in and go to the admin dashboard.
  2. Click Board Members in the admin navigation, or go directly to:
https://www.okanoganvalleygolf.com/admin/board-members

This page lists all current board members in display order.

Screenshot suggestion: /admin/board-members showing the list of board members with photo thumbnails, name, title, and Edit/Delete buttons.

Add a Board Member

  1. Click Add Board Member.
  2. Fill in the form:
    • Name — The member's full name.
    • Title — Their board role (e.g., President, Treasurer).
    • Email — Optional contact email. If left blank, the club's main contact email is used as the default.
    • Photo — Optional headshot (JPEG, PNG, GIF, or WebP; max 5 MB).
    • Sort Order — Controls the display order on the public page. Smaller numbers appear first.
  3. Click Create Board Member.
Screenshot suggestion: the Create Board Member form with all fields visible, including the photo upload area.

Edit a Board Member

  1. From /admin/board-members, click Edit next to the member.
  2. Update any fields:
    • To replace the photo, upload a new file — the old one is removed automatically.
    • To remove the photo without replacing it, click the Remove Photo button if shown.
  3. Click Save Changes.

The photo field on the edit screen is always empty. This is normal — uploading a new file replaces the existing one. Leave it blank if you do not want to change the photo.

Delete a Board Member

  1. Click Delete next to the board member and confirm.

The record and any uploaded photo are removed immediately. There is no undo.

Deletion is permanent. If you want to temporarily hide a board member, consider removing them from the public page by checking whether a display toggle exists, or simply delete and re-add them when needed.

Tips & Gotchas

Use sort order to control the display sequence.
Board members are listed by sort_order (smallest first). Assign numbers like 10, 20, 30 so you can insert someone between two existing members later without renumbering everything.

Photos are optional.
Members without a photo will display a placeholder. You can add a photo later by editing the record — you do not need to have one ready when first creating the entry.

Email defaults to the club contact email.
If a board member's email is left blank, the club's main contact email (from Theme Settings) is used wherever their email appears. Fill it in if they have a preferred personal or board email address.

Check the public page after changes.
After saving, open /board-members in a new tab to confirm the order and details look correct.
