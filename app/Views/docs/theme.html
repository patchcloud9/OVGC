Theme Settings — Colors, Branding & Layout

Theme Settings is where you control the overall look of the website — the color scheme, the club logo and favicon, contact information used site-wide, and a few layout preferences. Changes here take effect immediately for all visitors.

This is also where template variables get their values
When any page uses {{name}}, {{email}}, {{phone}}, {{address1}}, {{address2}}, or {{city}}, the actual text comes from the fields on this page. Update them once here and they update everywhere.

Getting to Theme Settings

  1. Log in to the admin panel.
  2. In the top navigation, click Admin (or go directly to /admin).
  3. Click Theme in the admin menu, or go directly to /admin/theme.

The page has three sections: Color Palette, Branding Assets, and Layout Options. All three are saved together with a single Save Theme Settings button.

Color Palette

Every color field works the same way: a color picker square on the left and a text box showing the hex code on the right. Click the square to open a color chooser, or type a hex code directly. Both stay in sync automatically.

What is a hex color code?
A six-character code starting with # that represents a color. #ffffff is white, #000000 is black. You can find hex codes using any online color picker or by picking from the color square.

Keeping the navbar readable
The navbar text color and navbar background color need good contrast. If your background is dark, use a light text color like #ffffff. If the background is light, use a dark text color like #222222.

Hero Background Color vs. Homepage hero
This color field controls the hero banners on subpages (Rates, Contact, etc.). The homepage hero has its own separate background color and image settings — those are managed under Admin → Homepage.

Homepage Logo

The homepage logo appears in the navigation bar at the top of every page.

  1. Click Choose logo… under Homepage Logo.
  2. Select a PNG, JPG, or SVG file from your computer (max 2 MB).
  3. Click Save Theme Settings.

A preview of the current logo is shown below the upload field once one has been saved.

Recommended size: 200 × 50 px
The logo is displayed at a fixed height in the navbar. Very tall or square images may be cropped or look oversized. A wide, horizontal logo at roughly 4:1 aspect ratio works best.

SVG files
SVG logos are accepted and ideal for logos because they stay sharp at any size. If your SVG file contains embedded scripts or unusual markup, the browser may not render it — use a clean export from a design tool.

Secondary Logo

The secondary logo field is available for storing an alternate logo (for example, a dark-background version or a small icon-only version). It is saved to the server but not currently displayed anywhere on the site. You can upload one now to have it ready for future use.

Favicon

The favicon is the small icon that appears in browser tabs and bookmarks next to the site name.

  1. Click Choose favicon… under Favicon.
  2. Select an ICO or PNG file (max 2 MB).
  3. Click Save Theme Settings.

Recommended size: 32 × 32 px or 16 × 16 px
Favicons are displayed very small. Use a simple, high-contrast icon — a logo with fine text will not be legible at tab size. A square PNG at 32 × 32 px is the most compatible choice.

Saving Your Changes

  1. Make your changes anywhere on the form.
  2. Scroll to the bottom and click the green Save Theme Settings button.
  3. The page reloads and a green confirmation bar appears at the top.

All three sections are saved together
There is one save button for the entire form. You do not need to save Color Palette, Branding, and Layout separately.

Reset to Defaults

At the bottom of the page, below the save button, there is a Reset Theme to Defaults button inside a yellow warning box.

Clicking it will:

  • Restore all colors to their original values (primary blue-purple, accent green, etc.)
  • Remove the saved logo and favicon paths (the uploaded files are not deleted from the server, but the theme will stop using them)
  • Reset the header to Static and the card style to Default

This cannot be undone
The reset happens immediately when you confirm the browser prompt. Your custom colors and logo selections will be cleared. The uploaded image files remain on the server, so you can re-upload them, but you will need to re-enter all color values manually if you want them back.

Tips & Gotchas

Change the club name and contact info first
Before customizing colors or uploading a logo, fill in Site Name, Contact Email, Phone, and Address. These power the {{template}} variables used in homepage text, and missing values will leave blank gaps on the live site.

Test your color choices
After saving, visit a few public pages to check that text is readable against your chosen navbar and button colors. Low-contrast combinations are easy to miss in the admin form but obvious to visitors.

Logo file size limit is 2 MB, not 5 MB
The theme logo and favicon have a stricter 2 MB limit (compared to the 5 MB limit for homepage images). If your upload silently fails, check that the file is under 2 MB.

Hero Background Color only applies to subpages
The homepage hero has its own independent background color and image settings under Admin → Homepage. The "Hero Background Color" field here only affects the dark banner at the top of pages like Rates, Contact, and Board Minutes.

Secondary Logo is stored but not yet displayed
Uploading a secondary logo saves the file to the server for future use. It does not currently appear anywhere on the public site. You can safely skip this field.
