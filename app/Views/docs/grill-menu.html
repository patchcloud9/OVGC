Grill Menu

The Grill Menu feature lets you upload the clubhouse food menu as a PDF. When you upload a PDF, a preview image is automatically generated and displayed on the public /menu page. Visitors can view the menu inline and download the PDF with one click. Swapping the menu when prices or items change takes about 30 seconds.

Getting to the Grill Menu Admin

  1. Log in and go to the admin dashboard.
  2. Click Grill Menu in the Pages section, or go directly to:
https://www.okanoganvalleygolf.com/admin/grill-menu

The page shows the current status of the PDF and the auto-generated preview image, along with the upload form.

Uploading or Replacing the Menu

  1. Go to /admin/grill-menu.
  2. Click Choose File and select the menu PDF from your computer.
  3. Click Upload PDF.

After the upload completes, the server automatically converts the first page of the PDF into a preview image. You will see one of two messages:

"Menu PDF uploaded and preview image generated successfully."
Everything worked. The public menu page is now showing the new menu as an image with a Download PDF button.

"PDF uploaded, but automatic image conversion failed."
The PDF was saved and the Download button works, but the inline image preview could not be generated. Visitors will see View Menu and Download buttons instead of the image. This is rare and typically means the image conversion software is temporarily unavailable on the server.

The admin page shows a thumbnail of the current preview image so you can confirm it looks right before leaving.

Screenshot suggestion: /admin/grill-menu showing the Current Status section with a green "Uploaded" PDF tag, a green "Generated" image tag, and the thumbnail preview below.

Removing the Menu

If you need to take the menu offline temporarily:

  1. Scroll to the Remove Menu section at the bottom of the page.
  2. Click Remove Menu and confirm.

Both the PDF and the preview image are deleted immediately. The public /menu page will show a "check back soon" message until a new PDF is uploaded.

Removal is immediate. Keep a copy of your menu PDF on your computer before removing it, in case you need to re-upload it later.

The Public Menu Page

The public menu page is at /menu. Depending on what has been uploaded, it shows one of three things:

  • Preview image + Download PDF button — when both the PDF and the auto-generated image are present. This is the normal state after a successful upload.
  • View Menu + Download buttons — when a PDF was uploaded but image conversion was not available. Both buttons open/download the PDF directly.
  • "Check back soon" message — when nothing has been uploaded yet, or the menu was removed.
Screenshot suggestion: the public /menu page showing the menu image with the Download PDF button in the top-right corner.

Adding the Menu to the Navigation

The /menu link is not added to the navigation automatically. To make it visible in the site nav:

  1. Go to Menu Management in the admin dashboard.
  2. Add a new top-level item with the title Menu (or Grill Menu) and URL /menu.
  3. Set the display order to place it where you want it in the navbar.
  4. Save.

Tips

Keep the original PDF on your computer.
The admin page does not let you download the current PDF (the public page does). Keep your source file saved locally so you can re-upload it or make edits easily.

Max file size is 10 MB.
Standard single-page menu PDFs are usually well under 1 MB. If your PDF is large, try exporting it from your design software at a lower resolution or using a free online PDF compressor.

Only the first page is shown as the preview image.
If your menu is multiple pages, only page 1 will appear inline. All pages are included in the downloaded PDF. If you have a two-sided menu, consider combining both sides into a single-page landscape layout before uploading.

Uploading a new PDF replaces the old one instantly.
There is no draft or preview step — the new menu goes live as soon as the upload completes. If you need to update prices mid-season, just upload the new PDF and the page updates immediately.
