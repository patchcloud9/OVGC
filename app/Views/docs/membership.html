Membership

Use this page when you need to update the /membership page (membership tiers, pricing, and introductory text) or manage the Board of Directors listing at /board-members. Both sections are fully editable from the admin panel — no code changes required.

How Memberships Are Organized

The public membership page is built from three layers:

LayerWhat It IsExample
Page ContentIntro text, member benefits bullet list, and a closing paragraph at the bottom of the page"Membership entitles you to…"
Membership GroupsCategory headings that organize related tiers togetherStandard, Lifetime, Under 30
Membership ItemsIndividual price lines within a group — each has a name and a priceSingle $750, Couple $1,200

Groups and items can be reordered and toggled active/inactive. Only active groups appear on the public page.

Manage Membership Groups

Create a Group

  1. Go to /admin/membership.
  2. Click Add Membership Group.
  3. Fill in the form:
    • Slug — A short machine name with no spaces (e.g., standard, under-30). Used internally only.
    • Title — The visible group heading on the public page (e.g., Standard Memberships).
    • Subtitle — Optional second heading line shown below the title.
    • Note — Optional description paragraph for the group.
    • Sort Order — Controls the group's position on the page. Smaller numbers appear first.
    • Active — Set to 1 to publish, 0 to hide without deleting.
  4. Click Create Group.

If a validation error occurs (e.g., slug already in use or title missing), the form will stay open with your input preserved so you can correct only what needs fixing.

Edit or Delete a Group

  1. From /admin/membership, click Edit next to the group.
  2. Update any fields and click Save Changes.
  3. To remove the group entirely, click Delete and confirm.

Deleting a group also deletes all its membership items. There is no undo. If you want to temporarily hide the group, set Active = 0 instead.

Manage Items within a Group

Each group contains one or more membership items — the individual price lines (e.g., Single, Couple). To manage a group's items, click Items next to the group on the /admin/membership page, or go to:

https://www.okanoganvalleygolf.com/admin/membership/{groupId}/items
Screenshot suggestion: the group items page showing a list of price lines with their name, price, notes, and Edit/Delete buttons.

Add an Item

  1. From the group items page, click Add Item.
  2. Fill in:
    • Name — The tier label (e.g., Single, Couple, Junior).
    • Price — The price displayed (e.g., $750, $1,200). Enter exactly as you want it to appear.
    • Sort Order — Controls the item's position within the group. Smaller numbers first.
    • Notes — Optional fine-print text shown beneath the price line.
  3. Click Create Item.

Edit or Delete an Item

  1. Click Edit next to the item.
  2. Update the fields and click Save Changes.
  3. To remove it, click Delete and confirm.

Update the Membership Page Text

The top of the admin membership page has a Page Content form for the text that frames the pricing tables. Changes here affect the public /membership page immediately after saving.

FieldWhere it appears
Top TextIntroductory paragraph shown above all membership groups
BulletsMember benefits list — one benefit per line; displayed as bullet points on the public page
Bottom TextClosing paragraph shown below all groups (good for a call to action or contact prompt)
  1. Edit the text fields as needed.
  2. Click Save Page Content.

One bullet per line. Each line you type in the Bullets field becomes a separate bullet point on the public page. Press Enter to start a new bullet. Blank lines are ignored.

Board of Directors

The board member listing at /board-members is managed separately from membership pricing. Go to:

https://www.okanoganvalleygolf.com/admin/board-members
Screenshot suggestion: /admin/board-members showing the list of board members with their photo thumbnails, name, title, and Edit/Delete buttons.

Add a Board Member

  1. Click Add Board Member.
  2. Fill in:
    • Name — The member's full name.
    • Title — Their board role (e.g., President, Treasurer).
    • Email — Optional contact email. If left blank, the club's main contact email is used as the default.
    • Photo — Optional headshot (JPEG, PNG, GIF, or WebP; max 5 MB).
    • Sort Order — Controls the display order. Smaller numbers appear first.
  3. Click Create Board Member.
Screenshot suggestion: the Create Board Member form with all fields visible, including the photo upload area.

Edit a Board Member

  1. Click Edit next to the board member.
  2. Update any fields.
    • To replace the photo, upload a new file — the old one will be removed automatically.
    • To remove the photo without replacing it, use the Remove Photo button if shown.
  3. Click Save Changes.

Delete a Board Member

  1. Click Delete next to the board member and confirm.

The record and any uploaded photo are removed immediately.

Tips & Gotchas

Use active/inactive to stage changes.
Set a group to Active = 0 to prepare a new pricing tier without it appearing publicly. Flip it to 1 when you're ready to publish.

Sort order controls everything.
Both groups and items are ordered by their sort_order value (smallest first). If two items share the same sort order number, their relative order is not guaranteed — use unique numbers to keep control.

Deleting a group cascades to its items.
All membership items inside a deleted group are also deleted. There is no recovery. If you only want to hide the group, set Active = 0.

Price is free-form text.
Enter prices exactly as you want them displayed (e.g., $750 or $750.00/year). There is no currency formatting applied automatically — be consistent across all items.

Board member photos are optional.
Members without a photo will display a placeholder or no image, depending on the theme. You can add a photo later by editing the record.

Check the public page after changes.
After saving, open /membership and /board-members in a new tab to confirm everything looks correct before the next visitor sees it.
