Event Flyers

The Flyers feature lets you upload and display promotional flyers for upcoming events. Active flyers appear on the public /flyers page as a mini-gallery. You can also link a flyer directly to a specific event so it shows up on that event's detail page. Flyers expire automatically — each one has an expiry date (default 90 days from upload) after which it disappears from the public page.

Getting to the Flyers Admin

  1. Log in and go to the admin dashboard.
  2. Click Flyers in the Pages section, or go directly to:
https://www.okanoganvalleygolf.com/admin/flyers

The list shows all flyers — active and expired. Each row shows a thumbnail preview, title, expiry date, status badge, and Edit / View / Delete buttons.

Status badges:
Active — Visible on the public flyers page.
Expiring Soon — Expires within 7 days.
Expired — Past the expiry date; hidden from the public page but still in the list here.

Screenshot suggestion: /admin/flyers showing the table with a mix of active and expired flyers, including thumbnail previews and status badges.

Add a Flyer

  1. From /admin/flyers, click Add Flyer.
  2. Fill in the form:
    • Title (required) — A short name for the flyer, e.g. "Summer Tournament".
    • Description (optional) — Shown below the title on the public page. Keep it brief.
    • Flyer File (required) — An image or PDF. Accepted formats: JPG, PNG, GIF, WebP, PDF. Max 10 MB.
    • Expires On (required) — The date the flyer stops appearing publicly. Defaults to 90 days from today. You can extend it at any time by editing the flyer.
  3. Click Upload Flyer.

Image flyers (JPG, PNG, etc.) are shown inline as cards on the public page. PDF flyers display a document icon with a "View PDF" link instead.

Screenshot suggestion: the Add Flyer form showing the Title, Description, file picker, and Expires On date fields.

Edit a Flyer

You can update the title, description, or expiry date without touching the file. You can also swap the file entirely.

  1. Click the Edit (pencil) button next to the flyer.
  2. The current file is shown at the top as a preview — confirm you have the right flyer.
  3. Update whichever fields you need:
    • Change Title or Description as needed.
    • To extend an expired flyer, update Expires On to a future date — it will become active again immediately.
    • To replace the file, choose a new one under Replace File. Leave it blank to keep the existing file.
  4. Click Save Changes.

Extending expiry is the preferred way to keep a flyer visible.
Rather than deleting and re-uploading, just edit the expiry date. The flyer's link to any events it is attached to is preserved.

Screenshot suggestion: the Edit Flyer form showing the current image preview, the title/description fields, and the Expires On date input.

Delete a Flyer

  1. Click the Delete (trash) button next to the flyer and confirm.

The database record and the file on the server are both removed immediately. If the flyer was linked to any events, those events will no longer show a flyer on their detail page. There is no undo.

Deletion is permanent. If you might need the flyer later, consider extending its expiry date and leaving it in the list instead of deleting it.

Linking a Flyer to an Event

You can attach a flyer to a specific event so it appears in a sidebar on that event's public detail page.

  1. Go to Events in the admin panel (/admin/events).
  2. Click Edit on the event you want to link.
  3. Scroll down to the Linked Flyer section at the bottom of the form.
  4. Select a flyer from the dropdown. A preview appears immediately so you can confirm it is the right one.
  5. Click Save Changes.

To remove the link, select — No flyer — from the dropdown and save.

The flyer sidebar only appears on the public event detail page when the linked flyer is active (not expired). If the flyer expires, the sidebar disappears automatically — no action required. The link is preserved in the database, so extending the flyer's expiry date will make it visible again.

Screenshot suggestion: the event edit form showing the Linked Flyer box with a flyer selected and the image preview visible.

The Public Flyers Page

Active flyers are shown at /flyers as a card grid. Each card displays:

  • The flyer image (or a PDF icon for PDF files).
  • The title and optional description.
  • A Full Size link (images) or View PDF link (PDFs) to open the file directly.

To add a link to this page in the site navigation, go to Menu Management and add a child item under the Events dropdown with URL /flyers.

Screenshot suggestion: the public /flyers page showing two or three flyer cards with titles and images.

Tips & Gotchas

Set the expiry date to match the event date.
A flyer for a July 4th tournament should expire on or just after July 4th. The 90-day default is a starting point — adjust it to fit the specific event.

Max file size is 10 MB.
If an upload fails, the error message will describe the reason. Large images can be compressed in an image editor before uploading. PDFs can be reduced using "Save as optimised PDF" options in most PDF tools.

Expired flyers stay in the admin list.
They are not deleted automatically. You can revive one by editing its expiry date, or clean up old ones manually with the Delete button.

The event-flyer link survives expiry.
If a linked flyer expires, the event detail page simply stops showing the sidebar. Extend the expiry date and the sidebar comes back — you do not need to re-link.

Field Reference

FieldRequiredNotes
TitleYesDisplayed on the public flyers page and in admin. Min 2 characters, max 255.
DescriptionNoShort text shown below the title on the public page. Max 1000 characters.
Flyer FileYes (upload) / No (edit)JPG, PNG, GIF, WebP, or PDF. Max 10 MB. On the edit screen, leave blank to keep the existing file.
