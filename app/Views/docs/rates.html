Rates

Use this page when you need to update or review the [rates page](/rates) (green fees, cart rentals) on the public site, and manage price groups in the admin panel.

Public Rates Page (Visitor View)

URL: /rates

Visitors can see:

  • Rate group headings (Golf Fees, Cart Rentals, etc.)
  • Line items with a description and price
  • Optional text blocks above or below the rates table
  • Rules/bullet list text
Screenshot suggestion: public /rates page with at least one group, rate lines, and a rules section.

Admin Rate Group Management

URL: /admin/rates

This page is for admin users only. You can:

  • Create, edit, delete rate groups (categories)
  • Set active/inactive status for showing/hiding on public page
  • Adjust group sort order
  • Edit the public page text (rules + above/below text)
  • Upload or replace a scorecard image
Screenshot suggestion: /admin/rates page showing group list and action buttons.

Create a Rate Group

  1. Go to /admin/rates.
  2. Click the Create Rate Group button.
  3. Fill in:
    • Slug (machine name, e.g. golf-fees).
    • Title (visible heading on /rates).
    • Subtitle (optional additional heading text).
    • Note (optional description).
    • Sort Order (smaller numbers appear first).
    • Active set to 1 to publish.
  4. Click Save.

If you get a validation error, the page will show a message and keep your input so you can adjust the fields.

Edit or Delete a Rate Group

  1. From /admin/rates, click the group name or Edit action.
  2. Change values and click Save Changes.
  3. To remove a group, click Delete and confirm. This also removes all rates in that group.

Deleting is permanent. If you want to hide a group without losing data, set Active= 0 instead.

Manage Rates within a Group

After selecting a group, go to /admin/rates/{groupId}/rates.

Actions available:

  • Add Rate (new line item)
  • Edit (modify description, price, notes, sort order)
  • Delete (remove rate line)
Screenshot suggestion: group rates page with list of lines and Add/Edit/Delete controls.

Add a New Rate

  1. Click Add Rate.
  2. Provide Description, Price, Sort Order, and optional Notes.
  3. Click Create Rate.

Edit a Rate

  1. Click Edit next to the rate.
  2. Change values and click Save.

Delete a Rate

  1. Click Delete on the rate row.
  2. Confirm the deletion prompt.

Update the public rates page text

The main /admin/rates page also has fields for page content:

  • Rules Text – appears as bullet lists on /rates.
  • Above Rates – content shown before rate groups.
  • Below Rates – content shown after rate groups.
  • Scorecard Upload – optional image file (JPG/PNG) used by the scorecard link.
  1. Enter or edit text fields.
  2. Upload scorecard using the file selector (only JPG/PNG).
  3. Click Save.

Tips and reminders

  • Use active/inactive to control visibility without deleting group settings.
  • Check public /rates after changes to confirm layout and order.
  • Work with the pro shop manager before changing published rates.
